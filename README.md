# Weather App

A modern weather application built with React, Vite, and Tailwind CSS.

## Features
- Search for weather by city
- Real-time weather updates
- Current conditions and forecasts
- Responsive design

## Tech Stack
- React
- Vite
- Tailwind CSS
- Weather API

## Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
